# The Sunny-Side: Hoskins Family Reunion Landing Page

A standalone luxury family reunion website built with Next.js, Tailwind CSS, Framer Motion, and React Hook Form.

## Project Identity

- Brand: The Sunny-Side: Hoskins Family Reunion
- Theme: Prestigious Family Legacy
- Catchphrase: Rooted in Love. Growing for Generations.
- Tech Stack: Next.js 15, React 18, Tailwind CSS 3, Framer Motion 11, React Hook Form

6. Connect section with itinerary timeline, GroupMe, family tree, and required form
7. Footer with quick external links

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start local development:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
npm start
```
